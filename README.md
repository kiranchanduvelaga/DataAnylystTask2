#  Excel Pivot Table Analysis 

##  Project Overview
This project is  focused on analyzing a sales dataset using Excel Pivot Tables.
The reports help users quickly understand sales performance, customer behavior, regional trends, and profitability across different dimensions such as Region, Category, and Customer Segment.
The goal of this task is to summarize sales data, identify business insights, and create an interactive pivot-based report.

---

##  Dataset Used
- Dataset Name: Superstore Sales Dataset  

---

## Tools Used
- Microsoft Excel / Google Sheets  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- Conditional Formatting  

---

## Task Objectives
- Clean and prepare raw sales data
- Create pivot tables for sales and profit analysis
- Identify top-performing categories and regions
- Build interactive dashboards using slicers
- Write meaningful business insights

---

## Process Followed

### 1. Data Cleaning
- Removed blank rows and incorrect entries
- Ensured correct data types for dates, sales, and profit
- Standardized category, region, and segment names

### 2. Data Preparation
- Converted dataset into Excel Table format
- Added a calculated column:
  - Profit Margin = Profit / Sales

### 3. Pivot Table Analysis
Created multiple pivot tables:
- Total Sales by Category
- Sales by Region and Segment
- Average Profit Margin by Category and Region

### 4. Visualization
- Created pivot charts to visualize sales distribution
- Applied conditional formatting to highlight performance
- Added slicers for Region, Category, and Segment

### 5. Insights & Reporting
- Wrote 5 summarized business insights based on pivot analysis
- Exported final pivot report as PDF

---

## Key Insights
- West region contributes the highest share of total sales
- Technology category generates the highest revenue
- Certain regions show low profit margins despite high sales
- Consumer segment dominates overall sales
- Some categories require cost optimization to improve margins
